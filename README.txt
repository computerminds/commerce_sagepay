INTRODUCTION
------------

This module allows Drupal Commerce customers to pay using the Sagepay Form integration.
https://www.sagepay.co.uk/support/12/36/sage-pay-form

INSTALLATION
------------

 1. Download and copy the 'commerce_sagepay' folder into your modules directory.

 2. Enable the module under /admin/modules within the group 'Commerce (contrib)'.

 3. Add a new Payment gateway under 'admin/commerce/config/payment-gateways'
    You will need your Sagepay Vendor Name and encryption keys for your
    Test and Live environments.
